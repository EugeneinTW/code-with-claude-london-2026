# Code with Claude London 2026

**Anthropic 倫敦開發者大會 · 2026 年 5 月 19–20 日 · 倫敦**

這份文件是 [Code with Claude London 2026](https://eugeneintw.github.io/code-with-claude-london-2026/) 全場 24 堂演講的互動式學習筆記。以「The Claude Dispatch」雜誌風格製作，每份筆記包含結構化整理、關鍵洞見、給初學者的概念解說、互動視覺圖表與思考題。

## 關於這場大會

Anthropic 開發者大會（Code with Claude）匯聚全球頂尖 AI 工程師、研究者與產品負責人，示範如何以 Claude 構建下一代智慧應用。2026 年首次移師倫敦，為期兩天，共 24 場演講。
